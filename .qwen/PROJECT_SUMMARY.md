# Project Summary

## Overall Goal
Create a comprehensive GitHub-powered bidirectional markdown note-taking app with offline capabilities that supports both LTR and RTL languages, featuring real-time rendering, multi-file management, and theme switching.

## Key Knowledge
- **Project Name**: NotesHubBidi (also referred to as Markdown Renderer)
- **Technology Stack**: HTML5, CSS3, JavaScript (ES6+), Bootstrap 5.3, Bootstrap Icons, Marked.js, Highlight.js, KaTeX, Vite
- **Key Features**: Real-time markdown rendering, multi-file tab system, theme switching (light/dark), bidirectional text support (LTR/RTL), enhanced code blocks with copy/collapse, LaTeX math support, PWA with offline capabilities
- **Architecture**: Client-side only application with local storage persistence, modular JavaScript with dedicated rendering module, CSS custom properties for theming
- **Build Commands**: `npm run dev` for development, `npm run build` for production, `npm run preview` for previewing production build
- **File Structure**: Main files include index.html, script.js, src/markdown/renderer.js, style.css, with additional assets in public/ directory
- **Special Focus**: Bidirectional text support for Persian/Arabic content while maintaining general markdown functionality
